# Фриланс

При работе на Upwork клиенты учитывают следующее:
- Цена
- Отзывы
- Предложение 
- Портфолио

Самое сложное — начать работу с пустым профилем. Можно попробовать провести заказ через старого клиента, чтобы получить первый отзыв.
